# Checkers
client/server desktop checkers game

## Technologies Used
- Java 17
- JavaFX 19
- Hibernate 5
- JUnit 5

## Features
- 3 different checkers variants
- Playing against another player or a simple bot
- Saving a game to the database
- Retrieving the game from the database
